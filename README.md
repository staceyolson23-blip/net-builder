# 3D Net Generator

An interactive tool for math students to enter dimensions for 3D solids and get:
- A **printable net** with glue tabs and a scale bar
- An **animated 3D fold** showing how the net becomes the solid

## Solids supported
- Rectangular prism
- Cylinder
- Square pyramid
- Triangular prism
- Cone

## How to use
1. Select a solid type and units
2. Enter your dimensions
3. Click **Print net** to print at 100% scale
4. Click **Fold into 3D** to see the animation
